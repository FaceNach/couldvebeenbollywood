# Could've Been Bollywood 🎬🌎

**Could've Been Bollywood** is a fun, interactive web application that calculates and visualizes the statistical probability of your birth country, and lets you "re-roll the dice of life" to see where you would have been born in an alternate reality based on global population data.

This project features a striking **Retro / Neubrutalism** design aesthetic with heavy borders, hard shadows, vibrant colors, and custom pixelated cursors.

## 🚀 Features

- **Check Your Odds:** Input your actual country of origin to see the exact statistical probability of being born there.
- **Roll the Dice of Life:** Click the big blue button to run a weighted random algorithm that selects a new country based on its real-world population size.
- **Dynamic Data & Visuals:** Automatically fetches live data (Capital, Population, Region) using a country API.
- **Interactive Maps:** Built with `react-simple-maps`, it dynamically auto-centers and zooms perfectly into your newly generated country.
- **Retro Aesthetic:** Implements a Neubrutalist design with hover/active animations and custom cursors, built on top of Tailwind CSS and shadcn/ui.

## 🛠️ Technologies Used

- **Framework:** React 19 + Vite
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4 + Custom Vanilla CSS (for cursors and animations)
- **Components:** shadcn/ui (Radix Primitives)
- **Maps:** `react-simple-maps` & `d3-geo` (for projection and dynamic zooming/centering)
- **Data Source:** Custom weighted random algorithm + REST Countries API data integration.

# Could've Been Bollywood 🎬🌎 (Español)

**Could've Been Bollywood** es una aplicación web interactiva y divertida que calcula y visualiza la probabilidad estadística de tu país de nacimiento, y te permite "volver a tirar los dados de la vida" para ver dónde habrías nacido en una realidad alternativa basada en datos de población mundial.

Este proyecto cuenta con una llamativa estética de diseño **Retro / Neobrutalista** con bordes gruesos, sombras duras, colores vibrantes y cursores pixelados personalizados.

## 🚀 Características

- **Revisa tus Probabilidades:** Ingresa tu país de origen actual para ver la probabilidad estadística exacta de haber nacido ahí.
- **Tira los Dados de la Vida:** Haz clic en el botón azul gigante para ejecutar un algoritmo aleatorio ponderado que selecciona un nuevo país basándose en el tamaño real de su población.
- **Datos y Visuales Dinámicos:** Obtiene automáticamente datos en tiempo real (Capital, Población, Región) utilizando una API de países.
- **Mapas Interactivos:** Construido con `react-simple-maps`, centra y hace zoom automáticamente a la perfección sobre tu país recién generado.
- **Estética Retro:** Implementa un diseño Neobrutalista con animaciones al pasar el mouse/hacer clic y cursores personalizados, construido sobre Tailwind CSS y shadcn/ui.

## 🛠️ Tecnologías Utilizadas

- **Framework:** React 19 + Vite
- **Lenguaje:** TypeScript
- **Estilos:** Tailwind CSS v4 + CSS puro (para cursores y animaciones)
- **Componentes:** shadcn/ui (Radix Primitives)
- **Mapas:** `react-simple-maps` y `d3-geo` (para la proyección, el centrado y el zoom dinámico)
- **Fuente de Datos:** Algoritmo propio de aleatoriedad ponderada + Integración con datos de API de países.
